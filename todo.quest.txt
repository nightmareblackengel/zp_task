Разработать чат :slight_smile:.

Основные требования:

выбор технологий - свободный (всё что душе угодно можно использовать)

регистрация - не нужна, вход свободный - по вводу имейла

юзер зашёл - остальные его сессии протухли

максимально быстрый обмен сообщениями

количество одновременных подключений около 20 - 1000

чат должен поддерживать наличие “каналов” и приватных сообщений (1 на 1)

поддержка истории до [1 / 7 / 30 дней | 500 / 1000 / 5000 сообщений] в зависимости от настройки на пользователе / канале. Внимание! Если 2 пользователя ведут диалог и у одного из них настройка 30 дней, а у второго 7 дней - история хранится по максимальному сроку / количеству сообщений.

Любой из чатов должен поддерживать предварительно заданный набор специальных команд:

/date - показывает текущую дату

/me some string - отправляет в чат “USERNAME some string” в формате, отличном от стандартного сообщения

/showmembers - работает только в каналах - выводит список пользователей, которые добавлены в канал

/kick {email} то отлично
/clearhistory - работает только для канала, после выполнения текущему пользователю отправляет количество удалённых сообщений
/sendwithdelay {N} {message} - отправляет в общий чат или канал сообщение {message} через {N} секунд*
    (мое мнение: можно реализовать через SendTimeout javascript)
